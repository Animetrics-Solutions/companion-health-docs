# 1. PRINCIPLE

The interface must feel like a system, not a collection of components.

- Calm
- Precise
- Deterministic

---

# 2. TYPOGRAPHY SYSTEM

## 2.1 Primary Typeface (Identity Only)
Faire Octave

Use ONLY for:
- Hero headlines
- Section anchors
- Brand moments

Weights:
- Regular
- Medium

---

## 2.2 Secondary Typeface (System UI)
Suisse Intl

Use for:
- UI labels
- Data
- API
- Navigation
- Supporting text

Weights:
- Light (secondary info)
- Regular (default)
- Medium (emphasis)

---

## 2.3 Rules

- Never mix serif + sans in same component
- Serif = narrative / identity
- Sans = system / function
- Numeric values always use Suisse

---

# 3. COLOR SYSTEM

## 3.1 Primary
#0E3E15 (Deep Green)

Use ONLY for:
- Active states
- Risk indicators
- CTA elements

---

## 3.2 Secondary Palette
- Sage (#B1DAB9)
- Blue-grey (#B5CFD5)
- Neutral (#E6E5DF)

---

## 3.3 Rules

- Color must indicate meaning
- Never use color decoratively
- Limit palette usage per screen (max 2 colors + neutral)

---

# 4. SPACING SYSTEM

## 4.1 Base Unit
8px grid system

---

## 4.2 Spacing Scale
- 8px (micro)
- 16px (tight)
- 24px (default)
- 40px (section spacing)
- 64px (major separation)
- 96px (hero spacing)

---

## 4.3 Rules

- Prefer more space over more elements
- Avoid visual crowding
- Maintain vertical rhythm consistency

---

# 5. COMPONENT PHILOSOPHY

## Avoid:
- Heavy cards
- Strong borders
- Boxed UI

## Prefer:
- Floating elements
- Spatial grouping
- Contrast-based hierarchy

---

# 6. GRID SYSTEM (DETAILED)

## 6.1 Layout Grid

- 12-column grid
- Max width: 1280px
- Margin (left/right): 80px desktop

---

## 6.2 Column Structure

- Columns: 12
- Gutter: 24px
- Column width: flexible

---

## 6.3 Breakpoints

Desktop: 1280+
Tablet: 768–1279
Mobile: <768

---

## 6.4 Vertical Grid

- Baseline: 8px
- Align all text + components to baseline
- Maintain vertical rhythm

---

## 6.5 Alignment Rules

- All elements must snap to grid
- No floating misalignment
- Consistent left alignment across sections

---

## 6.6 Grid Visibility

- Subtle grid overlay allowed
- Opacity: <5%
- Never compete with content

---

# 7. VISUAL ELEMENTS

- Use brand wave patterns sparingly
- Avoid clutter
- Maintain calm visual field

---

# 8. CONSTRAINTS

Do NOT:
- Over-design
- Over-color
- Over-animate

---

# 9. SUCCESS CRITERIA

- Feels like a system
- Not visually noisy
- Immediate clarity

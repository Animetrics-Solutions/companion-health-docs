# 1. PRINCIPLE

The grid enforces order, rhythm, and clarity.

All UI must align to grid.

---

# 2. BASE GRID

Unit: 8px

All spacing and sizing must be multiples of 8.

---

# 3. LAYOUT GRID

## Desktop

- Width: 1280px
- Columns: 12
- Gutter: 24px
- Margin: 80px (left/right)

---

## Tablet

- Columns: 8
- Gutter: 20px
- Margin: 40px

---

## Mobile

- Columns: 4
- Gutter: 16px
- Margin: 16px

---

# 4. COLUMN BEHAVIOR

- Flexible width
- Consistent alignment
- No arbitrary positioning

---

# 5. VERTICAL RHYTHM

- Baseline: 8px
- All text + components aligned

---

# 6. SPACING GRID

Allowed spacing:

8 / 16 / 24 / 40 / 64 / 96

---

# 7. ALIGNMENT RULES

- Left alignment preferred
- Avoid center-heavy layouts
- Maintain visual flow

---

# 8. GRID OVERLAY

- Optional visual grid
- Opacity: 3–5%
- Used for:
  - alignment
  - depth

---

# 9. COMPONENT ALIGNMENT

- All elements must snap to grid
- No floating offsets
- No visual drift

---

# 10. BREAKPOINT BEHAVIOR

- Maintain structure across devices
- Reduce complexity on smaller screens

---

# 11. SUCCESS CRITERIA

- Clean alignment
- Predictable spacing
- No visual chaos

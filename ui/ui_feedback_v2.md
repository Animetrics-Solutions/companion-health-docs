# UI Feedback — Companion Health V2
Scope: UI System Enforcement
Status: CRITICAL

Applies to:
- Homepage
- Login
- System UI

---

# 1. OVERALL ISSUE

UI is visually strong but not fully system-compliant.

Primary problem:

The interface still behaves partially like:
- a landing page
- a composed UI

Instead of:
- a continuous system surface

---

# 2. GRID SYSTEM — MAJOR GAP

## Problem

Grid is not visible or consistently enforced.

---

## Required Fix

- Enforce strict 12-column grid
- All elements must snap to grid
- No floating alignment

---

## Vertical Rhythm

- 8px baseline
- spacing must use:
  24 / 40 / 64 only

---

## Outcome

UI must feel:
- precise
- engineered
- mathematically aligned

---

# 3. TYPOGRAPHY ENFORCEMENT

## Rules

- Faire Octave → ONLY:
  - Hero
  - Section anchors

- Suisse Intl → ALL UI

- Suisse Intl Mono → labels, API, metadata

---

## Critical Fix

- Risk numbers → Suisse Intl Medium
- Remove serif from ALL system UI

---

## Outcome

System feels:
- computational
- not editorial

---

# 4. COMPONENT SYSTEM

## Problem

Residual card / container patterns

---

## Remove

- cards
- box containers
- shadows

## Replace With

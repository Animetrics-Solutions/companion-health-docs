# 1. CORE VERDICT

The demo is visually strong and technically refined.

However:

It still behaves like a **designed experience**, not a **natural system state**.

---

# 2. PRIMARY ISSUE — “CONTROLLED SYSTEM”

Even with ambient updates, the system feels:

- orchestrated
- too clean
- too perfectly staged

User perception:

“This is simulating reality”

NOT:

“This is reality”

---

# 3. RISK ENGINE ISSUE (CRITICAL)

## Current

Risk randomly fluctuates:

72 → 69 → 76 → etc

---

## Problem

This breaks credibility.

Risk should NOT feel random.

---

## Fix

Replace random fluctuation with:

- directional drift
- state-based logic

Example:

- stable → slight increase → plateau
- not oscillation

---

## Rule

Risk must feel:

computed, not animated
